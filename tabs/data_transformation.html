Trello Data Transformation

Python

For work my team and I analyze a large amount of Trello data. If you're not familiar with Trello it is a workflow collaboration tool that is used as a todo list to track tasks and their progress. For more information you can check out their website here for more info.

As our dataset grew, Excel was not capable of handling all the data. As a result I decided to create scripts that would not only consolidate multiple boards but transform the data within. What would take our team hours or weeks to perform now completes these tasks in seconds. To clone or check out this repo you can check it out on my Githubhere.

File Consolidator

When exporting a Trello team, Trello downloads a zipped folder with each board in a sub folder. The contents should be extracted into the data folder within this repo. If you are trying to merge a couple of files Excel may be just fine, but in our case we needed to merge over 500 files. The initial version of this script took ~30-45 minutes to merge all the files and after a few iterations it now takes ~1 minute.
